## [面试题 16.16. 部分排序](https://leetcode-cn.com/problems/sub-sort-lcci/)

**analyse:**

```
找出3段，然后二分
```

**code:**

```python
class Solution:
    def subSort(self, array: List[int]) -> List[int]:
     
        if not array: return [-1, - 1]
        m, n = 1, len(array) - 2
        while m < n and array[m] >= array[m - 1]:
            m += 1
        while n >= 0 and array[n] <= array[n + 1]:
            n -= 1
        if m == len(array) or n < 0: return [-1, -1]
        
        max = array[m]
        min = array[m]
        for i in range(m, n + 1):
            if array[i] > max:
                max = array[i]
            if array[i] < min:
                min = array[i]
        
        m -= 1
        n += 1

        l, r = 0, m
    
        while l < r:
            mid = l + r >> 1
            if array[mid] > min:
                r = mid
            else:
                l = mid + 1
        m = r
        l, r = n, len(array) - 1
        while l < r:
            mid = l + r + 1 >> 1
            if array[mid] < max:
                l = mid
            else:
                r = mid - 1
        n = r
        return [m, n]

```
