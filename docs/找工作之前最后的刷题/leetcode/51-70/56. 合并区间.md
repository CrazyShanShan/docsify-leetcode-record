## [56. 合并区间](https://leetcode-cn.com/problems/merge-intervals/)

**代码：**

```java
class Solution {
    public int[][] merge(int[][] a) {
        int n = a.length;
        int m = a[0].length;
        List<int[]> res = new ArrayList<>();
        
        // 排序
        Arrays.sort(a, (o1, o2)->{
            return o1[0] - o2[0]; 
        });
        int st = a[0][0], ed = a[0][1];
        for (int i = 1; i < n; i++) {
            if (a[i][0] > ed) {
                res.add(new int[]{st, ed});
                st = a[i][0];
                ed = a[i][1];
            } else ed = Math.max(ed, a[i][1]);
        }
        res.add(new int[]{st, ed});
        
        return res.toArray(new int[res.size()][2]);
    }
}
```
