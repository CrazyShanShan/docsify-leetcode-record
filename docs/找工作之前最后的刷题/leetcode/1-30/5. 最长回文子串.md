## 5. 最长回文子串

**分析：**

简单的DP问题。

**代码：**

```java
class Solution {
    public String longestPalindrome(String ss) {
        char[] s = ss.toCharArray();
        int n = s.length;
        
        // dp
        boolean[][] dp = new boolean[n][n];
        int left = -1, right = -1;
        int len = 0;
        for (int i = n - 1; i >= 0; i--) {
            for (int j = n - 1; j >= i; j--) {
                if (i == j) dp[i][j] = true;
                else if (j - i == 1) dp[i][j] = s[i] == s[j];
                else dp[i][j] = s[i]== s[j] && dp[i + 1][j - 1];
                if (dp[i][j] && j - i + 1 > len) {
                    len = j - i + 1;
                    left = i;
                    right = j;
                }
            }
        }
        return ss.substring(left, right + 1);
    }
}
```
