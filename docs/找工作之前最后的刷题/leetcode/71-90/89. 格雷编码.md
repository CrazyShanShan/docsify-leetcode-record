## [89. 格雷编码](https://leetcode-cn.com/problems/gray-code/)

**代码：**

```java
class Solution {
    public List<Integer> grayCode(int n) {
        List<Integer> res = new ArrayList<>();
        res.add(0);
        while (n-- > 0) {
            for (int i = res.size() - 1; i >= 0; i--) {
                res.set(i, res.get(i) << 1);
                res.add(res.get(i) + 1);
            }
        }
        return res;
    }
}
```
