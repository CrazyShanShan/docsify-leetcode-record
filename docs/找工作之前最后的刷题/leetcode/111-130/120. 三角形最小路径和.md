## [120. 三角形最小路径和](https://leetcode-cn.com/problems/triangle/)

**analyse:**

```
dp下到上
```

**code:**

```java
class Solution {
    public int minimumTotal(List<List<Integer>> t) {
        int n = t.size();
        int m = n + 1;
        int[][] dp = new int[2][m];
        for (int i = n - 1; i >= 0; i--) {
            for (int j = 0; j <= i; j++) {
                int num = t.get(i).get(j);
                if (i == n - 1) dp[i & 1][j] = num;
                else {
                    dp[i & 1][j] = Math.min(dp[i + 1 & 1][j], dp[i + 1 & 1][j + 1]) + num;
                }
            }
        }
        return dp[0 & 1][0];
    }
}
```
