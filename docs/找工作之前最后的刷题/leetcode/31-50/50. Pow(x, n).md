## [50. Pow(x, n)](https://leetcode-cn.com/problems/powx-n/)

**分析：**

```
快速幂。
```

**代码：**

```java
class Solution {
    public double myPow(double x, int n) {  
        boolean isMinus = false;
        if (n < 0) isMinus = true;
        double res = 1;
        for (long k = (long)Math.abs((long)n); k > 0; k >>= 1) {
            if ((k & 1) == 1) {
                res *= x;
            }
            x *= x;
        }
        if (isMinus) res = 1 / res;
        return res;
    }
}
```
