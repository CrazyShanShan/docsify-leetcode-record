---

- 数组

  1. 长度必须开始指定
  2. 保存的类型必须同一类型
  3. 增加/删除元素 消耗时间复杂度比较高

- 集合

  1. 可以动态保存多个对象，使用比较方便
  2. 提供了一系列方便的操作对象的方法： 增删改查
  3. 使用集合添加、删除新元素的代码，比较简洁
